# Improvements Lighthouse

Project dedicated to improvements and tracking of various feature engineering and importance labels.

## Charts Interaction Notes

- Double-click no longer resets zoom.
- Per-chart zoom reset is available in `Actions` -> `Reset Zoom`.
- Page-wide zoom reset is available in `Actions` -> `Reset Zoom All Plots`.
- Hovering any chart shows a synchronized vertical hover line across visible charts.
- Click on chart area drops a pinned cursor (max 5 per page, session-only).
- Pinned cursors can be dragged; dragging off left/right chart edge removes the pin.
- Bottom 20% of chart plot area acts as a pan zone (`drag` equivalent to `Shift + Drag`).
