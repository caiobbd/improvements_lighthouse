# Phase 7 Plan 02: Frontend Incremental Fetch and SWR Cache

## Objective

- **What:** Implement per-tag cache keys, in-flight dedupe, and incremental merge fetch pipeline.
- **Why:** Full-batch cache keys and full reload behavior waste requests and degrade UX.
- **Output:** Stale-while-revalidate UX with selective fetches and atomic merge per chart.
