# Phase 8 Plan 01: Global Cursor State and Interaction Arbitration

## Objective

- **What:** Extend chart runtime architecture with a page-global cursor model (hover + pins) and deterministic interaction arbitration.
- **Why:** Hover/pin synchronization must be reliable across charts without breaking existing zoom/pan/drag behaviors.
- **Output:** Stable cursor-state contract shared by all cards on active page, with strict click-vs-drag distinction.
