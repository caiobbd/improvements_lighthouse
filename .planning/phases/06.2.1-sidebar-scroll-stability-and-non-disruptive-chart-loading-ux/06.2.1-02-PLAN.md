# Phase 06.2.1 Plan 02: Non-Disruptive Chart Loading UX

## Objective

- **What:** Refactor chart loading/render behavior so charts remain visible and contextual while data fetch runs.
- **Why:** Operators should never lose visual context during tag additions or refreshes.
- **Output:** Persistent chart canvas + non-destructive loading overlays for empty and populated charts.
