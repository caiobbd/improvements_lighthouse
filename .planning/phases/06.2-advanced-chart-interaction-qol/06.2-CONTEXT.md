# Phase 6.2: Advanced Chart Interaction QoL - Context

## Decisions

- Drag interactions on charts default to marquee zoom.
- Modifier keys for drag behavior are locked:
  - `Ctrl+drag`: X-axis-only zoom
  - `Alt+drag`: Y-axis-only zoom
  - `Shift+drag`: horizontal pan only (Y locked)
  - `Ctrl+Shift+drag`: synced X zoom selection across all charts on active page
- Synced X zoom must show gray preview span on every chart while selection is in progress.
- Wheel-zoom modifiers are locked:
  - `Ctrl+wheel`: X-axis zoom
  - `Alt+wheel`: Y-axis zoom
- Browser/page zoom must be prevented whenever pointer is inside chart area, including while timeseries is loading.
- Reset behaviors are both required:
  - double-click chart resets zoom/pan
  - `Reset View` entry in chart `Actions`
- Pan is clamped to loaded-data range (no background fetch in this phase).
- Split Y-axis behavior:
  - left side only, color-matched labels
  - only available with up to 4 sensors
  - action disabled with tooltip when unavailable
  - if active and sensor count becomes >4, auto-disable + notice
- Normalization behavior:
  - normalize all visible traces to `0..1`
  - tooltip shows normalized + original value
  - split-Y is disabled while normalization is active
- Charts with no visible points still receive synced X-domain updates so future data aligns with current page view.

## Discretion Areas

- Exact placement and visual style of the per-chart `?` help icon as long as it stays near Refresh and remains discoverable.
- Interaction-state storage location (chart-local state vs shared page-scoped state) as long as synchronization and reset rules are deterministic.
- Rendering optimization details (throttle/debounce/requestAnimationFrame) as long as zoom/pan remains responsive and stable.
- Notice mechanism for auto-disabling split axis (toast, inline hint, or temporary badge) as long as feedback is explicit.

## Deferred Ideas

- Trackpad/touch gesture tuning and momentum behavior.
- Pan-triggered incremental data fetching beyond current loaded range.
- Per-chart selective sync groups (sync only selected charts).
- Multi-axis right-side labeling variants and advanced axis layout manager.
